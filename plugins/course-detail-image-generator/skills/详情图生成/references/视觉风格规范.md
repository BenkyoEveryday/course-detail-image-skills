# 视觉风格规范

本规范从已提供的课程商品封面、详情页首图和介绍图中提炼。它用于在生成前选定方向，不把任何单张参考图当作固定模板。

## 总原则

- 先读需求，再选方向：由学段、学科、课题、商品类型和 Word 素材决定视觉语言。
- 一张图只建立一个主叙事：场景、主物、人物、标题共同服务同一课题，避免无关装饰拼贴。
- 标题是主视觉的一部分：使用具有笔触、手写、书法、厚重块面、描边、纸张或金属质感的定制字效；标题应与场景形成前后景关系，不能退化为普通系统字加色块。
- 图文留白服务层级：大标题先被看见，卖点第二，素材展示第三；细节信息不抢主标题。
- 所有文字仍须严格取自 Word。没有提供的文字，保持留白，不补造标签、角标、口号、英文小字或底部品牌条。

## 学段气质

### 小学

- 关键词：亲近、探索、想象、游戏、温暖、自然。
- 人物：低龄儿童、圆润可爱比例、清晰表情和动作；动物、精灵、植物等可作为亲和引导者。
- 场景：森林、花园、校园、童话列车、雨后窗边、课本情境、轻幻想世界。
- 标题：圆润手写、童趣泡泡字、软边描边字、轻快笔刷字；高对比、易读但不僵硬。
- 版式：允许大面积标题与卡片化信息，画面明亮通透，素材展示可用圆角卡片、叠放课件或桌面俯视。

### 初中

- 关键词：成长、自主、故事、任务、探索、身份感。
- 人物：少年感学生、青年化角色、历史人物或任务角色；避免幼儿化比例与过度卖萌。
- 场景：电影感教室、雨景、山海、古城、博物空间、探险现场、社会真实场景。
- 标题：有力量的笔刷书写、利落粗字、手绘字形、杂志式排版或题材化字效；可用低饱和高级色、强明暗和材质感。
- 版式：封面可用海报、分镜、卷轴、任务板、影视框等叙事结构；介绍图强调清晰的章节层级和课件实拍感。

### 高中

- 关键词：思辨、体系、深度、效率、未来感、学术感。
- 视觉：更克制、更清晰，强调信息组织和概念力量；可用结构图、真实场景、实验/数据/文化意象。
- 标题：更精炼的字形设计、书写与现代排版结合、强网格与留白；避免小学化气泡字和过多卡通角色。
- 版式：封面突出知识命题或学术张力，介绍图突出方法、体系、证据和成果。

## 学科视觉语义

| 学科 | 可选主叙事 | 标题与画面方法 | 避免 |
|---|---|---|---|
| 语文 | 文本情境、四季自然、阅读空间、诗意想象、故事角色 | 书写字、留白、纸张/窗景/山水/植物，标题可有笔触与呼吸感 | 通用办公课件感、题材无关的卡通 |
| 历史 | 时空穿越、文物、古城、人物抉择、生活史、文明起源 | 时代材质、卷轴、碑刻、墨迹、文物纹样、电影光影；可用现代热词与古史反差做任务感 | 只堆古风纹样、没有叙事主体 |
| 地理 | 山海、地球、地图、野外观察、区域风貌、探索任务 | 大尺度景观、航拍/地形视角、地图纸、坐标和路线；粗体或地貌感字效 | 把地理只画成蓝色地球图标 |
| 道法 | 公共议题、法治、公民责任、成长选择、现实生活场景 | 纪实/任务式海报、议题符号、人物行动与清晰信息层级；可庄重也可有青年网络表达 | 空泛口号、说教式单一红色大字 |
| 科学 | 实验、观察、自然细节、实验室、微观世界、探索记录 | 透镜、标本、植物、星空、仪器、任务板；清晰可信的观察感 | 与主题无关的科技蓝光效 |
| 数学 | 解题探究、空间结构、现实应用、逻辑挑战 | 图形/坐标/建模作为场景线索，字效克制而有秩序 | 公式随机铺满背景 |
| 英语 | 语言情境、文化旅行、故事对话、生活任务 | 场景化口语、手账/旅行票据/故事书结构，标题可有国际化字形节奏 | 只用字母堆砌或国旗拼贴 |

## 封面与详情页首图

- 封面 `16:9` 和详情首图 `3:4` 都视为主视觉海报，不是普通信息页。
- 构图优先从以下方法中选一种：标题置于场景前景、标题在卷轴/黑板/纸牌/窗框中、人物与标题互动、场景包围标题、课件缩略图从下方或透视桌面延展。
- 首图必须有具体主题场景或插画配图，且与学段、学科相符。不能只用几何圆形、渐变色块和系统字体。
- 有素材展示时，把它们作为后景产品陈列或可读的卡片组，避免挤压标题主视觉。

## 后续详情介绍图

- 详情图固定 `3:4`。第 2 页起保持统一栏目系统：标题区位置、标题字效逻辑、强调色、说明文字样式、图片容器和页边留白相对一致。
- 可根据内容选用：两列对照、卡片矩阵、时间线/流程线、散点透视陈列、前后景叠放、纸张/文件层叠、设备屏幕展示、编号任务板。
- 每页只保留一个核心信息主题。正文可读性优先，配图应是论据或示例，不是装饰。
- 当展示 Word 中的课件截图、文件列表、表格、教案时，预留严格匹配的图片槽位；槽位可有边框、投影、编号和遮挡，但不在其中生成假文字。

## 选型速查

| 需求特征 | 优先方向 |
|---|---|
| 小学语文单元/阅读 | 自然童话、书本入口、圆润手写标题、儿童或动物引导 |
| 初中语文课文/诗文 | 电影感文本场景、书写题字、雨景/窗景/山海/古城等文学意象 |
| 初高中历史社团 | 时代探秘、生活史任务、卷轴/档案/文物、现代语境与古代叙事对照 |
| 初中地理 | 户外探索、地貌/地球/地图、粗字与坐标路线、规模感画面 |
| 初中道法 | 现实议题任务、纪实或戏剧化场景、庄重而有青年表达的标题 |
| 资料包/教案/任务单 | 首图仍用主题海报；介绍图转为清爽信息系统，以真实材料展示和流程解释为主 |

## 质量检查

- 能否在不读小字时，一眼看出学段、学科和课题？
- 标题有没有被设计成画面主角，而不是默认系统字？
- 场景、人物、插画和主题是否真的相关？
- 是否保留了 Word 指定图片的准确数量、比例和位置？
- 详情图后续页面是否同属一个设计系统，而非每页换一套风格？
- 是否误加了 Word 未给出的任何可读文字？
